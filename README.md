# Pomodoro Timer CLI

ターミナルで動くポモドーロタイマーです。

## 機能

- **作業セッション**: 25分
- **短い休憩**: 5分
- **長い休憩**: 15分（4セッション終了後）
- **操作**: 開始・停止・リセット・設定

## インストール

```bash
nix profile add github:nazozokc/pomodoro-cli?ref=v1.2.0
```

## 使い方

```bash
pomodoro
```

### メニュー

| アクション | 説明 |
|-----------|------|
| 開始 | タイ時停止中は再開マーを開始（一） |
| 停止 | タイマーを一時停止 |
| リセット | タイマーを初期状態に戻す |
| 設定 | 作業時間・休憩時間を変更 |
| 終了 | アプリを終了 |

### 設定項目

- 作業時間（デフォルト: 25分）
- 短い休憩（デフォルト: 5分）
- 長い休憩（デフォルト: 15分）
- セッション数（デフォルト: 4）

### オプション

```bash
pomodoro --version  # バージョンを表示
```
